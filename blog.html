Question-1: What are the different ways to select an element in the DOM?

We can select an element using getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), or querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML gets/sets HTML, innerText gets/sets visible text, and textContent gets/sets all text (including hidden ones).

Question-3: What is event delegation in the DOM?

Event delegation lets me attach a single event listener to a parent element to handle events for its children dynamically.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and propagates up through its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create elements using document.createElement(), add them with appendChild() or append(), and remove them using removeChild() or remove().
